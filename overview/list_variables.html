List of pyrad variables#

Note

Pyrad uses the following mappings to map the variable names in your files to the short names used by pyrad (dBZ, dBZc, RhoHV and else). the “Py-ART to Pyrad” mappings are used for CFRadial files and the “ODIM to Pyrad” mappings are used for ODIM HDF5 files. If your files do not contain the standard names listed below, you can use the keyword DataTypeIDInFiles in the loc files to provide them. Please see Location configuration file.
